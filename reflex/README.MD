## What is SABA Reflex?

Reflex는 SABA를 활용해 Physical Agent를 구축하는 프레임워크입니다. **Trigger**(언제?), **Action**(어떤것을), **Tools**(SABA devices)를 조합해서 자율적으로 작동하는 물리적 에이전트를 만들 수 있습니다.
